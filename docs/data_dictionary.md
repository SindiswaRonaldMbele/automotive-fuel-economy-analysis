# Data Dictionary
mpg - miles per gallon (fuel efficiency)
cylinders (no unit/quantity)- the number of engine cylinders
displacement (cu in) - engine displacement
horsepower (hp)- measure engine power output
weight (lb) - overall mass of the car
acceleration - the vehicle's acceleration performance
model year (1970-1982) - year the car was manufactured 
origin (1,2,3)- codes representing the country/region
car name - name of the car
